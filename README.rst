============
Nty Template
============

The default templates for setting up a Nty_ team.

.. _Nty: https://github.com/charlyoleg/nty


Getting started
===============

In a terminal::

  npx degit https://github.com/charlyoleg/nty-template team-xyz
  cd team-xyz
  npm i
  npm run create_desktop_icon
  npm run start_web_ui
